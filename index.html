  • 17: To Echo or Not to Echo

    Why do we sometimes need to "echo" the results of a function, and other times not? If the function returns the value, nothing will show unless you echo the results. However, if the function itself already echos something out, you won't need to echo the result when you call the function.

    In WordPress, we have the following functions:

    • the_title();
    • get_the_title();
    • the_ID();
    • get_the_ID();

    What's the difference? The rule is: If it says "get", WordPress simply gets the value for you; without "get," WordPress will echo out the result.
